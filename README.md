# TradeVault

A simulated capital markets trading and portfolio analytics platform.

## Tech Stack
- **Backend:** Java 21, Spring Boot 3, Spring Security, JPA/Hibernate
- **Frontend:** React 18, Axios, Recharts
- **Database:** PostgreSQL 16
- **Cache:** Redis
- **Queue:** RabbitMQ
- **Data:** Alpha Vantage API

## Features
- JWT authentication
- Real-time stock price lookup
- Simulated buy/sell order execution
- Portfolio holdings and P&L analytics
- Watchlist management
- Async trade processing via message queue
